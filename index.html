전략 한눈에 보기

3개의 레이어로 나뉘는 과금 구조

핵심 매출. 매칭 성사 시 양쪽 결제. 유일한 과금 순간. 코인/포인트 없음.
🍀
무료 리워드
클로버 (정성의 화폐)
비과금 리워드. 상대를 더 깊이 알아보는 데 사용. 퍼널 윤활유 역할.
🔮

클로버 소액 충전 (Phase 2)

만남권은 "결심의 순간"이고, 클로버는 "확신을 만드는 과정"입니다.
클로버가 확신을 쌓아주면, 만남권 결제 저항이 줄어듭니다.
두 레이어는 경쟁이 아니라 퍼널 위의 역할 분담입니다.
02

퍼널 × 과금 전략 맵

사용자 여정 6단계별 이탈 원인과, 만남권·클로버가 각각 어디서 작동하는지

STEP 1
발견

"가입비 없는 결정사?"

✓ STAY

Threads/인스타/네이버 광고 유입. "가입비 없는 결정사" 포지셔닝이 호기심 유발. 24문항 무료 분석이 진입점.

✕ BYE

결혼 의사 자체가 없거나, 데이팅앱과 차별화가 안 느껴짐. "결혼 앱? 관심 없어."

브랜드 전략
광고 메시지에서 "결정사"임을 명확히. "가입비 없는 결정사 · 가벼운 만남 앱이 아닙니다." 무료 24문항 분석이 리드 마그넷.
STEP 2
흥미

"웨이브 리포트 신기하네! 내 결이 이렇구나"

✓ STAY

24문항 완료 후 웨이브 시각화가 몰입 유발. Big Five 기반 분석이 자기 이해 욕구를 자극. 매칭 타입 결과에 대한 궁금증.

✕ BYE

"MBTI 같은 거 아냐?" 분석 신뢰도 의문. 24문항 귀찮아서 중도 이탈. 인증 과정의 진입 장벽.

🍀 클로버 역할
프로필 완성도 100% 달성 → 🍀 ×2, 24문항 완료 → 🍀 ×1 보상. "클로버를 모으면 매칭된 상대를 더 깊이 알아볼 수 있어요"라는 동기 부여로 온보딩 완주율 향상.
STEP 3
의심 해소

"신뢰할 수 있는 곳인가?"

✓ STAY

Trust Center(결혼중개업 신고필증, 보증보험, 환불규정), 인증 시스템, 첫 500매칭 인간 리뷰. 결정사 수준의 신뢰 인프라.

✕ BYE

"후기가 없네... 새 서비스라 불안해." 성공 사례 부재. "10만원인데 만남이 별로면?" 리스크 인식.

신뢰 전략
Trust Center를 온보딩 플로우에서 자연스럽게 노출. 매칭 플로우 안내 페이지에서 정찰제 ₩100,000 사전 안내 → 나중에 놀라지 않도록. 환불 정책 명시.
STEP 4
욕망

"이 사람 결이 나랑 맞다니! 만나보고 싶다"

✓ STAY

매칭 카드 도착 → 웨이브 리포트에서 결이 맞는 걸 확인 → 프로필이 매력적 → "만나볼래요"를 누르고 싶은 순간.

✕ BYE — 최대 이탈 구간

"결이 맞다는데... 10만원은 좀 부담이네." 욕망은 있지만 가격이 브레이크. "만남이 안 맞으면 돈 아깝다"는 리스크 인식.

🍀 클로버 역할 — 핵심 전환 구간
클로버는 이 갭을 메우는 퍼널 윤활유입니다.
상대방 전체 답변 열람(🍀×3), 확장 프로필 해금(🍀×2), 영상 프로필 열람(🍀×4), AI 궁합 코치(🍀×3), 웨이브 심화 분석(🍀×2) → 클로버를 쓸수록 "이 사람이 맞다"는 확신이 강해지고, 확신이 강할수록 ₩100,000 결제 저항이 줄어듭니다.
STEP 5
구매

"결이 맞았다! 서로 원했으니까 만나보자"

✓ STAY

양쪽 수락 확인 → "결이 맞았습니다!" 축하 → ₩100,000 결제. 서로의 진심 확인 후 결제하므로 "결심"으로 느껴짐.

✕ BYE

결제 화면에서 최종 이탈. 간편결제 미등록 시 마찰. 48시간 내 상대 미결제 → 자동 취소.

💰 만남권 결제
"만나볼래요"(무료) → "결이 맞았습니다!"(감정 피크) → "만남 시작하기"(₩100,000 결제)
결제가 욕망의 연장선에서 일어남. 거절 리스크 0인 상태에서 결제하므로 "돈 낭비" 불안 최소화. 간편결제(카카오페이/토스/애플페이) 필수.
STEP 6
재구매

"좋은 경험이었어. 다음 분도 만나볼까"

✓ STAY

만남이 좋았거나, 안 맞았더라도 "결을 찾는 과정"으로 긍정 해석. 피드백 기반 다음 매칭 품질 향상 약속.

✕ BYE

"한 번으로 충분해 / 만남이 별로였어." "10만원 낭비" 인식 → 신뢰 하락. 또는 성사되어서 불필요(최고 시나리오).

🍀 클로버 역할
만남 후 양쪽 피드백 작성 → 🍀×2 보상. 피드백이 다음 매칭 품질에 반영된다는 루프를 체감시킴. 매력 리포트(🍀×3)로 자기 프로필 개선 동기 부여.
💰 재구매 촉진 (Phase 2)
결이음권 묶음(3회 ₩270,000 / 5회 ₩400,000)으로 재구매 가격 저항 감소. 단, 런칭 시점에서는 단건 직접 결제만 운영.
03

결제 통합 플로우

현재 앱 매칭 플로우에 만남권 결제와 클로버를 삽입한 전체 여정

STEP 1 · 무료

매칭 카드 도착 — 프로필 확인

웨이브 리포트 + 기본 프로필. 가격 미노출 — 순수한 판단 공간 보호.

🍀 클로버 사용 가능 구간

상대 깊이 알아보기 (선택)

전체 답변 열람(🍀×3) · 확장 프로필(🍀×2) · 영상 프로필(🍀×4) · AI 궁합 코치(🍀×3) · 웨이브 심화(🍀×2). 사용할수록 확신 증가 → 구매 전환율 향상.

STEP 2 · 무료

"만나볼래요" 선택

버튼 하단 보조텍스트: "서로 만남 의사가 확인되면, 만남권(₩100,000)이 필요합니다"

대기

"만남의 의사를 전달하였습니다"

상대방 수락 대기. 현재 앱과 동일.

STEP 3 · 매칭 성사

🎊 "결이 맞았습니다!"

양쪽 모두 수락 확인 → 축하 화면. 감정 피크. 이 감정 위에 결제가 올라갑니다.

💰 STEP 3.5 · 결제 시점 — 유일한 과금 순간

"○○님과의 만남을 시작하시겠습니까?" — ₩100,000

양쪽 각각 결제. 양쪽 모두 완료 시 조율 단계 오픈. 48시간 내 미결제 시 자동 취소 + 전액 환불. 버튼 문구: "만남 시작하기" (결제하기 아님).

STEP 4 · 결제 완료 후

조율하기 — 일정 제안 → 확인 → 장소 선택

현재 앱 플로우와 동일. 양쪽 결제 완료 상태이므로 조율 진지도 향상.

🍀 클로버 사용 가능 구간

만남 준비 도우미 (선택)

첫인사 도우미(🍀×2) · 만남 준비 가이드(🍀×2). AI가 상대 성향 기반 대화 팁, 장소 추천, 주의점 제공.

STEP 5 · 만남 당일

안심번호 → 체크인 → 만남 → 피드백

현재 앱 플로우와 동일. 만남 후 피드백 작성 시 🍀×2 보상.

04

05

클로버(🍀) — 퍼널 윤활유 전략

과금과 완전 분리된 비과금 리워드. 상대를 더 깊이 알아가는 데만 사용.

클로버의 퍼널 역할
클로버는 직접 매출을 만들지 않습니다. 클로버가 하는 일은 STEP 3(의심 해소) → STEP 4(욕망) → STEP 5(구매) 전환율을 높이는 것입니다. 클로버를 써서 상대를 깊이 알수록 "이 사람 만나볼 만하다"는 확신이 강해지고, ₩100,000 결제 저항이 줄어듭니다.

수익 연결 포인트 (Phase 2)
AI 기능(궁합 코치, 첫인사 도우미, 매력 리포트, 만남 준비 가이드)은 실제 API 비용이 발생합니다. 향후 클로버 부족 시 소액 충전(₩3,000~5,000)을 열면, "정성의 화폐" 포지셔닝을 유지하면서 마이크로 수익 채널이 됩니다. 단, 런칭 시점에서는 클로버 구매 불가 — 활동 보상으로만 획득합니다.

가격 노출 전략

디자인 싱크 — 결정 사항

이번 미팅에서 합의할 것

Decision 1 — 결제 시점
양쪽 수락 후 결제(옵션 B)로 확정할 것인가?
→ 권장: 옵션 B. "결이 맞았습니다" 감정 피크 위에 결제.
Decision 2 — 네이밍 이원화
사용자 facing = 쉬운 말 / 내부 = 결이음으로 분리할 것인가?
→ 권장: 이원화. 결제 화면에서는 "만남 1회", 마케팅에서는 "결이음".
Decision 3 — 클로버 런칭 범위
8가지 사용처 중 런칭 시점에 어디까지?
→ 권장 MVP: 전체 답변 열람, 확장 프로필, 웨이브 심화 (비AI 3종). AI 기능은 Phase 2.
Decision 4 — 클로버 소액 충전
런칭부터 열 것인가, 추후 열 것인가?
→ 권장: Phase 2. 런칭 시 "정성의 화폐" 포지셔닝 확립 후 충전 옵션 추가.
Decision 5 — 양쪽 결제 타임아웃
48시간이 적절한가? 24시간? 72시간?
→ 권장: 48시간. 너무 짧으면 결제 압박, 너무 길면 상대 대기 부담.
Decision 6 — UX 카피 확정
"결이 맞았습니다!" / "만남 시작하기" 등 구체적 워딩 확정.
→ 디자인 팀과 A/B 테스트 후 결정.
"만나볼래요"는 마음의 표현 (무료).
"만남 시작하기"는 결심의 실행 (₩100,000).
이 두 순간 사이에 "결이 맞았습니다"라는 확인이 있기 때문에,
결제가 거래가 아니라 축하가 됩니다.
결하다 — 결이 맞아야, 결혼도 맞다.
